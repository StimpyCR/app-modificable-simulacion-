# ⚡ Inicio Rápido - UI Customizer

## 🚀 Los 3 Pasos Más Simples

### Paso 1: Instalar Dependencias
Abre una terminal en la carpeta del proyecto y ejecuta:
```bash
npm install
```

O simplemente **haz doble clic** en `install.bat`

### Paso 2: Ejecutar la App
```bash
npm start
```

O simplemente **haz doble clic** en `start.bat`

### Paso 3: ¡Personaliza!
- Cambia colores, fuentes, iconos desde el panel izquierdo
- Visualiza los cambios en tiempo real en la derecha
- Los cambios se guardan automáticamente

---

## 📋 Qué Puedes Cambiar

| Parámetro | Qué Hace |
|-----------|----------|
| **Nombre de la App** | El título que ves en la ventana |
| **Emoji/Icono** | El símbolo junto al nombre |
| **Color de Fondo** | Fondo principal de la app |
| **Color de Texto** | Color de todo el texto |
| **Color de Acento** | Bordes, acentos, detalles |
| **Tamaño de Fuente** | De 12px a 24px |
| **Radio de Borde** | Bordes redondeados (0px a 20px) |

---

## 🔍 Dónde Se Guardan los Cambios

Todos los cambios se guardan automáticamente en **`config.json`**

Puedes editar ese archivo manualmente si quieres, o usar la interfaz gráfica.

---

## 💡 Consejos

✨ **Usa el exportador**: Haz clic en "💾 Exportar Config" para guardar tu configuración
⚡ **Cambios instantáneos**: No necesitas reiniciar, todo es en tiempo real
🔄 **Restablecer**: Con un clic vuelves a valores por defecto
📱 **Prueba colores**: El previsualizador te muestra exactamente cómo se verá

---

## ❓ ¿Problemas?

Si vuelves a no tener las dependencias instaladas:
```bash
npm install
```

Si quieres empezar desde cero:
```bash
rm -r node_modules
npm install
npm start
```
